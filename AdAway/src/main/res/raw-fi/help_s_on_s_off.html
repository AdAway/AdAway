HTC-laitteet

AdAway ei toimi, kun käytät niin sanottua S-ON-laitetta. tämä 'ominaisuus' on monissa HTC-laitteissa ja estää AdAwayn kirjoittamasta hosts-tiedoston.

S-ON/S-OFF?

S-OFF tarkoittaa, että laitteen NAND-osa is lukittu ja voidaan kirjoittaa. HTC-laitteissa oletuksena on S-ON, mikä tarkoittaa, että myöskään sinä et voi käyttää tiettyjä järjestelmän alueita, eikä ole myöskään mahdollista varmistaa pysyvää pääkäyttäjäoikeuksia. Tämän lisäksi S-ON:lla allekirjoituksen tarkistaminen laiteohjelmiston otoksista on myös varmistettu.

Onko minulla S-ON tai S-OFF?

Boot into the Boot Menu on your device by holding down volume down button while pressing power and the text on top will show the flag status as either S-OFF or S-ON. A full root generally means S-OFF.

More information can be found on www.addictivetips.com. Additional S-OFF methods since Unrevokable (in link) might interest you, namely: Revolutionary, Revone, Firewater, RumRunner, Moonshine, SunShine...

Kiertotapa

Edellytys: Tietokoneellassi täytyy olla asennettuna toimiva Android SDK ADB-lisäosalla.

  1. Käynnistä laitteesi käynnistysvalikkoon painamalla äänenvoimakkuuden pienennys - ja virtapainiketta painettuna.
  2. Käytä Äänenvoimakkuus alas -nappia valitaksesi palautuksen.
  3. Clockworkissa valitse "partitions menu"
  4. Valitse mount /system, mount /sdcard ja mount /data
  5. Kytke usb-johto ja avaa komentorivi tietokoneeltasi.
  6. Enter adb shell and type ln -s /data/hosts /system/etc/hosts (This creates a symbolic link, which allows AdAway to edit the hosts file stored in /data while allowing the OS to use the file as if it were stored in /system.)
  7. Reboot your device and set Target hosts file to /data/hosts in AdAway's preferences.
  8. AdAway:n pitäisi toimia nyt.
