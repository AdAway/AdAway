Revenu publicitaire

Une idée fausse courante veut que si un utilisateur ne clique jamais sur les publicités, alors les bloquer est sans contrecoup financier pour le site ou l'application. C'est faux. Les développeurs gagnent de l'argent juste en affichant des publicités. Il n'en tient qu'à vous de bloquer ou non les publicités. Personnellement, je n'utiliserais pas d'applications affichant des publicités parce que je trouve cela très ennuyeux, alors sans blocage de publicité je n'installerais pas ces applications.

Comment fonctionne AdAway ?

AdAway utilise le fichier « hosts » pour bloquer les noms d'hôte qui servent des publicités. Le fichier « hosts », qui se trouve dans /system/etc/hosts, mappe les noms d'hôte vers des adresses IP. C'est une façon habituelle de définir des paires noms d'hôte-adresse IP sans se fier au système de noms de domaine (DNS). Tous les noms d'hôte non désirés sont redirigés vers 127.0.0.1 qui représente votre propre appareil.

Pourquoi l'option « Mise à jour automatique » journalière est-elle grisée ?

La vérification automatique de nouvelles mises à jour n'est proposée que si AdAway est installé sur la mémoire interne. Ceci est causé par des restrictions de la plate-forme Android.

Pourquoi dois-je redémarrer Android pour que jes changements prennent effet ?

Java sur Android maintient son propre cache DNS. Le système d'exploitation va immédiatement appliquer le nouveau fichier « hosts » (vérifiez-le avec ping depuis la ligne de commande) mais vous devrez redémarrer Android pour reconstruire le cache DNS de Java.

Comment utiliser le serveur Web d'AdAway ?

AdAway fonctionnera aussi sans utiliser le serveur Web !

Vous pouvez activer un serveur Web local dans les préférences d'AdAway pour répondre aux requêtes à l'adresse IP locale 127.0.0.1. Ainsi, le serveur Web d'AdAway répond aux requêtes des applications vers les serveurs de publicités qui sont redirigées vers 127.0.0.1.
Certaines applications refusent de fonctionner si les serveurs de publicités se sont pas joignables. Avec cette méthode, ils le seront à nouveau, répondant par une page blanche et sans image publicitaire.

Comment puis-je bloquer/débloquer certains noms d'hôtes ?

Ajouter les noms d'hôtes que vous voulez bloquer à la Liste noire (=Noire) sous Vos listes. De plus, les noms d'hôtes que voulez exclure du blocage peuvent être ajoutés à la Liste blanche (=Blanche) et, pour les noms d'hôtes que vous souhaitez rediriger vers une adresse IP spécifique, la Liste de redirection (=Redirection).
