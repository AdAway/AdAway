HTC eszközök

AdAway nem működik S-ON-t használó eszközön. Ez a 'képesség' számos HTC eszközön létezik, és megakadályozza az AdAway alkalmazást a hosts fájl írásában.

S-ON/S-OFF?

Az S-OFF azt jelenti, hogy az eszköz NAND rész nem zárolt, és írható. A HTC eszközök alapértéke S-ON, ami azt jelenti, hogy a rendszer bizonyos részeit nem érheti el és nem lehet állandó root jogot adni. Továbbá a firmware képek lenyomatának ellenőrzése is be van kapcsolva az S-ON jelző miatt.

Nálam S-ON vagy S-OFF van?

Boot into the Boot Menu on your device by holding down volume down button while pressing power and the text on top will show the flag status as either S-OFF or S-ON. A full root generally means S-OFF.

More information can be found on www.addictivetips.com. Additional S-OFF methods since Unrevokable (in link) might interest you, namely: Revolutionary, Revone, Firewater, RumRunner, Moonshine, SunShine...

Kerülő megoldás

Feltételek: telepíteni kell egy működő Android SDK-t a gépre egy ADB parancssorral együtt.

  1. Lépjen be az eszköze Boot menüjébe úgy, hogy a bekapcsoló gomb mellett a hangerő le gombot is nyomva tartja.
  2. Használja a Hangerő le gombot a visszaállítás mód választásához.
  3. A clockwork recoveryben mozogjon le a hangerő le gombbal a "partitions" menüre és érintse meg a kijelzőt a kiválasztáshoz.
  4. Válassza a mount/system, mount/sdcard és a mount/data lehetőségeket.
  5. Csatlakoztassa az USB összekötő vezetéket és nyisson meg egy parancssort a gépén.
  6. Enter adb shell and type ln -s /data/hosts /system/etc/hosts (This creates a symbolic link, which allows AdAway to edit the hosts file stored in /data while allowing the OS to use the file as if it were stored in /system.)
  7. Reboot your device and set Target hosts file to /data/hosts in AdAway's preferences.
  8. Az AdAway alkalmazásnak így már működnie kell.
