Dinheiro ganho com anúncios

Existe uma ideia errada muito repetida de que, se um utilizador nunca clicar em anúncios, bloqueá-los não vai prejudicar um site ou aplicação financeiramente. Isto é errado; os programadores também ganham dinheiro apenas por mostrar anúncios. Cabe-lhe a si decidir bloquear anúncios ou não. Pessoalmente, não utilizaria aplicações que mostram anúncios, visto que as acho muito irritantes, portanto, sem um bloqueador de anúncios, não instalaria estas aplicações.

Como funciona o AdAway?

O AdAway utiliza o ficheiro de hosts para bloquear hostnames que fornecem anúncios. O ficheiro de hosts é um ficheiro encontrado em /system/etc/hosts que mapeia hostnames para endereços IP. Esta é a forma tradicional de definir pares hostname-endereço IP sem recorrer ao Domain Name System (DNS). Os hostnames indesejáveis são redirecionados para 127.0.0.1, o que significa que apontarão para o seu próprio dispositivo.

Porque está a preferência de verificação diária de atualizações desativada?

Verificar atualizações automaticamente só está disponível quando o AdAway está instalado na memória interna do telemóvel. Isto deve-se a restrições da plataforma Android.

Porque tenho de reiniciar o Android para as alterações entrarem em efeito?

No Android, o Java mantém a sua própria cache interna de DNS. O sistema operativo refletirá o novo ficheiro de hosts imediatamente (verifique com um ping na linha de comandos), mas terá de reiniciar o Android para reconstruir a cache de DNS do Java.

Como utilizar o servidor web no AdAway?

O AdAway também funciona sem utilizar o servidor web!

Pode ativar um servidor web local nas preferências do AdAway para responder aos pedidos para o endereço IP local 127.0.0.1. Isto significa que os pedidos das aplicações para servidores web que forem redirecionados para 127.0.0.1 serão respondidos pelo servidor web do AdAway.
Algumas aplicações recusam-se a funcionar quando os servidores de anúncios não se encontram alcançáveis. Com este método, estes serão alcançáveis de novo, respondendo com uma página em branco sem imagens de anúncios.

Como posso bloquear/desbloquear hostnames específicos?

Adicione os hostnames que pretende bloquear à Lista negra (=Negra) em As suas listas. Além disso, os hostnames que pretende excluir do bloqueio podem ser adicionados à Lista branca (=Branca) e os hostnames que pretende redirecionar para um endereço IP específico pertencem à Lista de redirecionamento (=Redirec.).
