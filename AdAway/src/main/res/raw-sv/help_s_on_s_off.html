HTC Enheter

AdAway fungerar inte när du använder en enhet som har S-ON. Denna "egenskap" finns på många HTC enheter och hindrar AdAway från att skriva till hosts filen.

S-ON/S-OFF?

S-OFF betyder att NAND portionen av enheten är upplåst och kan skrivas till. Standardvärdet för HTC enheter är S-ON, vilket betyder att du inte har tillgång till vissa delar av systemet eller heller kan permanent root garanteras. Vidare, signatur checkar av firmware säkerställs även av S-ON flaggan.

Har jag S-ON eller S-OFF?

Boot into the Boot Menu on your device by holding down volume down button while pressing power and the text on top will show the flag status as either S-OFF or S-ON. A full root generally means S-OFF.

More information can be found on www.addictivetips.com. Additional S-OFF methods since Unrevokable (in link) might interest you, namely: Revolutionary, Revone, Firewater, RumRunner, Moonshine, SunShine...

Lösning

Förutsättning: Du måste installera en fungerade Android SDK med ADB på din PC.

  1. Boota in till Bootmenyn på din enhet genom att hålla in volym ner knappen samtidigt som du trycker in PåAv-Knappen.
  2. Använd volym ner för at välja recovery.
  3. I clockwork recovery, tryck volym ner till "partitions menu" och tryck på mittenknappen på telefonen för att välja
  4. Välj mount /system, mount /sdcard och mount /data.
  5. Sätt in usbsladden och öppna en kommandoprompt på din pc.
  6. Enter adb shell and type ln -s /data/hosts /system/etc/hosts (This creates a symbolic link, which allows AdAway to edit the hosts file stored in /data while allowing the OS to use the file as if it were stored in /system.)
  7. Reboot your device and set Target hosts file to /data/hosts in AdAway's preferences.
  8. AdAway ska nu fungera.
