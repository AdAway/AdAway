Inkomste deur advertensies

Daar bestaan 'n miskonsepsie dat indien die gebruiker nie op die advertensie klik nie, die blokkering, nie die webwerf of die program finansieel sal benadeel nie. Dit is egter nie waar nie, program ontwikkelaars verdien ook geld as die advertensie vertoon word. Dit is jou keuse of jy die advertensie wil blokkeer al dan nie. Persoonlik gebruik ek nie programme wat advertensie vertoon nie aangesien ek dit frustrerend vind, dus sal ek nie die programme sonder 'n advertensie blokkeerder installeer nie.

Hoe werk AdAway?

AdAway gebruik die gasheer lêer om gasheer name te blokkeer, wat advertensies diens. Die gasheer lêer is 'n lêer wat gevind word in /system/etc/hosts wat gasheer name aan IP adresse koppel. Dit is 'n tradisionele manier om gasheer name aan IP adresse te koppel sonder om op die Domein Naam Sisteem te vertrou (DNS). Alle ongewenste gasheer name word verwys na 127.0.0.1 wat beteken dit wys na jou eie toestel.

Hoekom is die daaglikse opdaterings blokkie grys?

Outomatiese nagaan vir opdaterings is slegs moontlik indien AdAway op die interne foon geheue geïnstalleer is. Dit is agv. beperkinge op die Android platvorm.

Hoekom moet Android weer begin om veranderinge te implementeer?

Java en Android onderhou hulle eie interne DNS geheue. Die bedryfstelsel sal dadelik nuwe gasheer lêers vertoon (verifieer dit met ping op die "command line") jy sal egter Android weer moet begin om die DNS geheue en Java te herbou.

Hoe om die webbediener in AdAway te gebruik?

AdAway sal ook werk sonder'n webbediener!

Jy kan 'n plaaslike webbediener aktiveer in AdAway voorkeure om versoeke vir 'n plaaslike IP adres te beantwoord 127.0.0.1. Dit beteken versoeke van programme aan advertensie bedieners wat herlei word na 127.0.0.1 word beantwoord deur AdAway se webbediener.
Sommige programme weier om te werk indien advertensie bedieners nie bereik kan word nie. Met hierdie metode is hulle weer bereikbaar en antwoord met 'n blanko bladsy en geen advertensies nie.

Hoe kan ek spesifieke gasheername blokkeer/toelaat?

Voeg die gasheer name wat jy wil blokkeer by jou Swartlys (=Black) onder Your Lists. Addisioneel kan jy gasheername wat jy wil uitsluit van uitsluiting byvoeg by jou Witlys (=White) en gasheer name wat jy wil herlei na 'n spesifieke IP adres by jou Herlei Lys (=Redirection).

Waar kan ek meer gasheer bronne vind?

Sien Lys van addisionele gasheer bronne vir AdAway.
