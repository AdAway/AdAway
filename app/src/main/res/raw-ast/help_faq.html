Entrada d'anuncios

Hai un conceutu erráu davezu que diz que si un usuariu enxamás fai clic nos anuncios, entós bloquialos nun firirá económicamente un sitiu o aplicación. Esto ye un enquívocu porque los desendolcadores tamién ganen dineru amosando namái anuncios. Depende de tigo si bloquies anuncios o non. Personalmente, nun usaría nin instalaría aplicaciones qu'amuesen anuncios ensin un bloquiador d'ellos porque alcuéntroles percafiantes.

¿Cómo furrula AdAway?

AdAway uses the hosts file to block hostnames, that serve advertisements. The hosts file is a file found in /system/etc/hosts that maps hostnames to IP addresses. It is a traditional way to define hostname to IP address pairs without relying on the Domain Name System (DNS). All unwanted hostnames are redirected to 127.0.0.1 which means they will point to your own device.

¿Por qué ta abuxáu'l caxellu d'anovamientu diariu?

La comprobación automática p'anovamientos nuevos ta namái disponible cuando Adaway ta instaláu na memoria interna del teléfonu. Esto ye pola mor de restricciones na plataforma Android.

¿Por qué teo de reaniciar Android pa que les camudancies faigan efeutu?

Java n'Andorid caltién la so caché DNS interna. El sistema operativu reflexará'l ficheru Hosts nuevu nel intre (verifícalo con «ping» na llinia comandos) pero necesitarás reaniciar Android pa reconstruyir la cahé DNS Java.

¿Cómo usar el sirvidor web n'AdAway?

AdAway will also work without using the webserver!

You can enable a local webserver in preferences of AdAway to answer requests to the local IP address 127.0.0.1. This means requests from applications to ad servers which are redirected to 127.0.0.1 are answered by AdAway's webserver.
Some applications refuse to work, when ad servers are not reachable. With this method they will be reachable again, replying with a blank page and no ad images.

¿Cómo pueo bloquiar/desbloquiar nomes d'agospiu específicos?

Add the hostnames you want to block to the Blacklist (=Black) under Your lists. Additionally, hostnames you want to exclude from blocking can be added to the Whitelist (=White) and hostnames you want to redirect to a specific IP address belong to the Redirect list (=Redirection).
