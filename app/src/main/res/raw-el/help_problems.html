Δεν αποκλείει τις διαφημίσεις στην εφαρμογή XYZ!

Μερικά ονόματα διακομιστών μπορεί να λείπουν στα παρεχόμενα από τις πηγές αρχεία hosts ή η εφαρμογή έχει ενσωματωμένες εικόνες για να εμφανίζει διαφημίσεις χωρίς πρόσβαση στο Internet.

Μπορείτε να καταγράψετε τις αιτήσεις DNS (Μενού->Καταγραφή αιτήσεων DNS) από το AdAway για να βρείτε ποια επιπλέον ονόματα διακομιστών πρέπει να φραγούν.

Προσθέστε τα ύποπτα ονόματα διακομιστών στη δική σας Μαύρη λίστα πατώντας παρατεταμένα τις εγγραφές στο αρχείο καταγραφών και επισημάνετε αυτά τα ονόματα, αφού τα επαληθεύσετε, στο Forum Hosts Inbox του hosts-file.net.

Αποθηκευμένες διαφημίσεις

Μερικές φορές οι εφαρμογές αποθηκεύουν τοπικά τις διαφημίσεις αφού τις μεταφορτώσουν. Αυτό οδηγεί σε παραμένουσες διαφημίσεις σε μερικές εφαρμογές. Μπορείτε να δοκιμάσετε να διαγράψετε την κρυφή μνήμη των εφαρμογών αυτών στη λίστα εφαρμογών του Android για να παρακάμψετε αυτό το πρόβλημα.

Some applications need to communicate with a hostname that is blocked by AdAway or refuses to work when the hostnames which should serve ads are not reachable. See https://github.com/AdAway/AdAway/wiki/ProblematicApps to get a list of known applications that have problems. Otherwise, find out which hostnames are needed and add them to your Whitelist under Your Lists and report them to the bug tracker of AdAway.
