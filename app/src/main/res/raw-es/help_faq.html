Ingresos por publicidad

Hay un error de concepto manifestado a menudo acerca de que si un usuario nunca hace clic en la publicidad, entonces bloquearla no dañará a un sitio o aplicación financieramente. Esto no es correcto, los desarrolladores también ganan dinero tan solo mostrando publicidad. Es cosa suya tanto si bloquea la publicidad como si no. Personalmente yo no usaría aplicaciones que muestren publicidad porque la encuentro muy molesta, así que sin un bloqueador de publicidad no instalaría estas aplicaciones.

¿Cómo funciona AdAway?

AdAway utiliza el fichero hosts para bloquear nombres de servidores que proporcionan publicidad. El fichero hosts es un fichero ubicado en /system/etc/hosts que mapea nombres de servidor hacia direcciones IP. Es una forma tradicional de definir pares de nombres de servidor a dirección IP sin apoyarse en el sistema de nombres de dominio (DNS). Todos los nombres de servidor no deseados se redirigen a 127.0.0.1, lo que significa que serán dirigidas de vuelta hacia su propio dispositivo.

¿Por qué está sombreada la preferencia de búsqueda de actualizaciones diarias?

La búsqueda automática de nuevas actualizaciones sólo está disponible cuando AdAway está instalado en la memoria interna del teléfono. Esto se debe a restricciones en la plataforma Android.

¿Por qué tengo que reiniciar Android para que los cambios surtan efecto?

Java sobre Android mantiene su propia caché interna de DNS. El sistema operativo reflejará el nuevo fichero hosts inmediatamente (verifique esto con un ping desde la línea de comandos) pero tendrá que reiniciar Android para reconstruir la caché de DNS de Java.

¿Cómo usar el servidor web en AdAway?

¡AdAway también funcionará sin usar el servidor web!

Puede habilitar un servidor web local en las preferencias de AdAway para responder peticiones a la dirección IP local 127.0.0.1. Esto significa que las peticiones desde aplicaciones hacia servidores de publicidad que sean redirigidas a 127.0.0.1, serán respondidas por el servidor web de AdAway.
Algunas aplicaciones se niegan a funcionar, cuando los servidores de publicidad no son alcanzables. Con este método volverán a ser alcanzables de nuevo, y les responderán con una página en blanco en lugar de imágenes publicitarias.

¿Cómo puedo bloquear/desbloquear nombres de servidor específicos?

Añada los nombres de servidor (host) que quiera bloquear a la Lista Negra (=Negra) bajo Sus listas. Adicionalmente, los nombres de servidor que quiera excluir del bloqueo pueden añadirse a la Lista Blanca (=Blanca), y los nombres de servidor que quiera redirigir a una dirección IP específica deben ir a la Lista de Redireccionamiento (=Redireccionamiento).
