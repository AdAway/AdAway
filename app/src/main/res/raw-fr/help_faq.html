Recettes publicitaires

Une idée fausse courante veut que si un utilisateur ne clique jamais sur les publicités, alors les bloquer est sans contrecoup financier pour le site ou l’application. C’est faux. Les développeurs gagnent de l’argent juste en affichant des publicités. Il n’en tient qu’à vous de bloquer ou non les publicités. Personnellement, je n’utiliserais pas d’applications affichant des publicités parce que je trouve cela très ennuyeux, alors sans blocage de publicité je n’installerais pas ces applications.

Comment AdAway fonctionne-t-il ?

AdAway utilise le fichier hosts pour bloquer les noms d’hôtes qui servent des publicités. Le fichier hosts se situe dans /system/etc/hosts et mappe les noms d’hôtes à des adresses IP. C’est une façon habituelle de définir des paires noms des serveurs/adresses IP sans avoir recours au système de noms de domaine (DNS). Tous les noms d’hôtes indésirables sont redirigés vers 127.0.0.1, c’est-à-dire votre propre appareil.

Pourquoi l’option de mise à jour quotidienne est-elle grisée ?

La vérification automatique de nouvelles mises à jour n’est proposée que si AdAway est installé sur la mémoire interne. Cela provient de restrictions de la plateforme Android.

Pourquoi dois-je redémarrer Android pour que jes changements prennent effet ?

Java sur Android maintient son propre cache DNS interne. Le système d’exploitation appliquera immédiatement le nouveau fichier hosts (vérifiez-le avec ping en ligne de commande), mais vous devrez redémarrer Android pour reconstruire le cache DNS de Java.

Comment utiliser le serveur Web d’AdAway ?

AdAway fonctionnera aussi sans utiliser le serveur Web.

Vous pouvez activer un serveur Web local dans les préférences d’AdAway pour répondre aux requêtes à l’adresse IP locale 127.0.0.1. Ainsi, le serveur Web d’AdAway répond aux requêtes des applications vers les serveurs de publicités qui sont redirigées vers 127.0.0.1.
Certaines applications refusent de fonctionner si les serveurs de publicités ne sont pas accessibles. Avec cette méthode, ils le seront de nouveau, répondant par une page blanche et sans images publicitaires.

Comment puis-je bloquer/débloquer certains noms d’hôtes ?

Ajoutez à la liste noire (=Noire) dans Vos listes les noms d’hôtes que vous voulez bloquer. De plus, les noms d’hôtes que voulez exclure du blocage peuvent être ajoutés à la liste blanche (=Blanche), et les noms d’hôtes que vous voulez rediriger vers une adresse IP précise peuvent être ajoutés à la liste des redirections (=Redirection).
