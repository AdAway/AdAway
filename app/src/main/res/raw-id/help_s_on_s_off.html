Perangkat HTC

AdAway tidak bekerja ketika Anda menggunakan perangkat dengan apa yang disebut S-ON. 'Fitur' ini ada pada banyak perangkat HTC dan mencegah AdAway menulis file host.

S-ON/S-OFF?

S-OFF berarti bahwa bagian NAND pada perangkat tidak terkunci dan dapat ditulis. Pengaturan standar untuk perangkat HTC adalah S-ON, yang berarti bahwa tidak dapat mengakses area tertentu dari sistem juga tidak bisa menjamin root permanen. Selanjutnya, memeriksa tanda tangan untuk gambar firmware juga dijamin oleh S-ON flag.

Apakah saya harus S-ON atau S-OFF?

Boot ke Menu Boot pada perangkat Anda dengan menekan tombol volume bawah </ i> sambil menekan tombol power dan keterangan di atas layar akan menampilkan status mungkin S-OFF atau S-ON. Sebuah root penuh umumnya bertanda S-OFF.

Informasi lebih lanjut dapat ditemukan di <ahref="http://www.addictivetips.com/mobile/what-is-s-off-how-to-gain-it-on-htc-android-phones-with-unrevoked-forever/">www.addictivetips.com. Metode S-OFF tambahan pada Unrevokable (di link) mungkin menarik bagi Anda, yaitu : Revolutionary, Revone, Firewater, RumRunner, Moonshine, SunShine...

Solusi

Prasyarat: Anda harus menginstal SDK Android dengan ADB shell pada PC Anda.

  1. Boot into the Boot Menu pada perangkat anda dengan menekan tombol volume down bersamaan dengan tombol power.
  2. Gunakan tombol volume down untuk memilih recovery.
  3. Pada volume pemulihan jarum jam ke "partisi menu" dan tekan trackpad untuk memilih
  4. Pilih mount /system, mount /sdcard and mount /data.
  5. Tancapkan USB mu dan buka terminal dalam PC mu.
  6. Masukkan kode berikut ke ADB shell ln -s /data/hosts /system/etc/hosts (Ini dilakukan untuk membuat link simbolik, yang memungkinkan AdAway untuk mengubah file hosts di /data dan tetap memungkinkan OS untuk menggunkan file tersebut seperti terletak di /system.)
  7. Matikan dan hidupkan kembali perangkat anda dan atur Target file host di prefrensi menjadi /data/hosts.
  8. AdAway seharusnya bekerja sekarang.
