Pendapatan oleh iklan

Terdapat kesalahpahaman yang sering menyatakan bahwa jika pengguna tidak mengklik iklan, kemudian memblokirnya tidak akan menyakiti keuangan situs atau aplikasi. Ini salah, pengembang juga memperoleh uang dengan hanya menampilkan iklan. Itu terserah kepada Anda apakah Anda memblokir iklan atau tidak. Secara pribadi saya tidak akan menggunakan aplikasi yang menampilkan Iklan karena saya menemukan mereka sangat mengganggu, sehingga tanpa pemblokir iklan saya tidak akan menginstal aplikasi ini.

Bagaimana cara kerja AdAway?

AdAway menggunakan berkas hosts untuk memblokir nama host, yang melayani iklan. Berkas host adalah berkas yang ditemukan di /system/etc/hosts yang memetakan nama host menjadi alamat IP. Ini adalah cara tradisional untuk menentukan nama host ke IP address pasangan tanpa bergantung pada Domain Name System (DNS). Semua nama host yang tidak diinginkan diarahkan ke 127.0.0.1 yang berarti mereka akan mengarah ke perangkat Anda sendiri.

Mengapa preferensi pengecekan pembaruan harian berwarna abu-abu?

Pemeriksaan otomatis pembaruan yang baru hanya tersedia bila AdAway diinstal pada memori telepon internal. Hal ini disebabkan oleh pembatasan pada platform Android.

Mengapa saya harus memulai ulang Android untuk perubahan efek?

Java pada Android mengelola DNS cache internalnya sendiri. Sistem operasi akan merefleksikan berkas host yang baru segera (memverifikasi dengan ping pada command line), tetapi Anda harus memulai kembali Android untuk membangun kembali cache DNS dari Java.

Bagaimana menggunakan webserver di AdAway?

AdAway juga akan bekerja tanpa memakai webserver!
Anda dapat mengaktifkan webserver lokal dalam preferensi AdAway untuk menjawab permintaan ke alamat IP lokal 127.0.0.1. Ini berarti permintaan dari aplikasi ke server iklan yang diarahkan ke 127.0.0.1 dijawab oleh webserver AdAway itu.
Beberapa aplikasi menolak bekerja, ketika server iklan tidak terjangkau. Dengan metode ini mereka akan dapat dijangkau lagi, menjawab dengan halaman kosong dan tidak ada gambar iklan.

Bagaimana saya memblokir/membuka blokir hostname tertentu?

Tambahkan nama host yang akan diblokir ke Daftar Hitam (= Hitam) di bawah Daftar Anda. Sebagai tambahan, nama host yang ingin Anda kecualikan dari pemblokiran dapat ditambahkan ke Daftar Putih (= Putih) dan nama host yang ingin Anda arahkan ulang ke alamat IP tertentu milik Daftar Pengalihan (= Pengalihan).

Di mana saya dapat menemukan lebih banyak sumber host?

Lihat Daftar sumber host tambahan untuk AdAway List of additional hosts sources for AdAway.
