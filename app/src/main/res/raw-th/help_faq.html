รายได้จากการโฆษณา

มีความเข้าใจผิดที่ระบุบ่อยๆว่าหากผู้ใช้ไม่เคยคลิกโฆษณาการปิดกั้นเนื้อหาดังกล่าวจะไม่ส่งผลเสียหายต่อไซต์หรือแอพพลิเคชันทางการเงิน ซึ่งเป็นสิ่งที่ผิดนักพัฒนายังมีรายได้ด้วยการแสดงโฆษณา ทั้งนี้ขึ้นอยู่กับคุณว่าคุณบล็อกโฆษณาหรือไม่ ส่วนตัวฉันจะไม่ใช้แอปพลิเคชันที่แสดงโฆษณาเนื่องจากพบว่าน่ารำคาญมากดังนั้นหากไม่มีตัวบล็อกโฆษณาฉันจะไม่ติดตั้งแอปพลิเคชันเหล่านี้

AdAway นี้ทำงานอย่างไร?

AdAway ใช้ ไฟล์โฮสต์ เพื่อป้องกันโฮสต์ที่ให้บริการโฆษณา ไฟล์โฮสต์คือไฟล์ที่อยู่ใน /system/etc/hosts ที่เชื่อมชื่อโฮสต์กับที่อยู่ IP โดยชื่อโฮสต์ที่ไม่ต้องการทั้งหมดถูกเปลี่ยนเส้นทางไปที่ 127.0.0.1 ซึ่งหมายความว่าลิงก์จะชี้ไปที่อุปกรณ์ของคุณเอง

เหตุใดการตั้งค่าการตรวจสอบการอัปเดตรายวันจึงเป็นสีเทา

การตรวจสอบอัตโนมัติสำหรับการอัปเดตใหม่จะใช้ได้เฉพาะเมื่อมีการติดตั้ง AdAway ไว้ในหน่วยความจำภายในเท่านั้น นี่เป็นเพราะข้อ จำกัด ของแพลตฟอร์มแอนดรอยด์

ทำไมถึงต้องรีสตาร์ทเพื่อให้มีผลเปลี่ยนแปลง?

Java บน Android มีแคช DNS ภายในของตัวเอง ระบบปฏิบัติการจะแสดงแฟ้ม hosts ใหม่ทันที (ตรวจสอบว่ามี ping ในบรรทัดคำสั่ง) แต่คุณจะต้องเริ่มต้นใหม่ Android เพื่อสร้างแคช DNS ของ Java

วิธีใช้เว็บเซิร์ฟเวอร์ใน AdAway

AdAway จะทำงานโดยไม่ใช้เว็บเซิร์ฟเวอร์!

คุณสามารถเปิดใช้งานเว็บเซิร์ฟเวอร์ในลักษณะที่ AdAway ต้องการเพื่อตอบรับคำขอไปยังที่อยู่ IP ในเครื่อง 127.0.0.1ซึ่งหมายถึงคำขอจากแอพฯที่ปกติจะส่งไปเว็บโฆษณาจะถูกเปลี่ยนเส้นทางไปยัง 127.0.0.1
ดังนั้นจะทำให้พวกโฆษณาจะไม่สามารถใช้งานได้

บล็อคหรือปลดบล็อคชื่อโฮสต์?

Add the hostnames you want to block to the Blacklist (=Black) under Your lists. Additionally, hostnames you want to exclude from blocking can be added to the Whitelist (=White) and hostnames you want to redirect to a specific IP address belong to the Redirect list (=Redirection).

ฉันสามารถหาแหล่งที่มาของโฮสต์ได้จากที่ไหน?

ดู รายชื่อแหล่งโฮสต์เพิ่มเติมจาก AdAway.
