รายได้จากการโฆษณา

มีความเข้าใจผิดที่ระบุบ่อยๆว่าหากผู้ใช้ไม่เคยคลิกโฆษณาการปิดกั้นเนื้อหาดังกล่าวจะไม่ส่งผลเสียหายต่อไซต์หรือแอพพลิเคชันทางการเงิน ซึ่งเป็นสิ่งที่ผิดนักพัฒนายังมีรายได้ด้วยการแสดงโฆษณา ทั้งนี้ขึ้นอยู่กับคุณว่าคุณบล็อกโฆษณาหรือไม่ ส่วนตัวฉันจะไม่ใช้แอปพลิเคชันที่แสดงโฆษณาเนื่องจากพบว่าน่ารำคาญมากดังนั้นหากไม่มีตัวบล็อกโฆษณาฉันจะไม่ติดตั้งแอปพลิเคชันเหล่านี้

AdAway นี้ทำงานอย่างไร?

AdAway ใช้ โฮสต์ไฟล์ เพื่อบล็อก รายชื่อโฮสต์, ที่ให้บริการโฆษณา ไฟล์ hosts คือไฟล์ที่พบใน/system/etc/hosts ที่ชื่อโฮสต์กับที่อยู่ IP.เป็นวิธีแบบดั้งเดิมในการกำหนดชื่อโฮสต์ไปยังคู่ IP แอดเดรสโดยไม่ต้องอาศัยระบบ Domain Name System (DNS). ชื่อโฮสต์ที่ไม่ต้องการทั้งหมดถูกเปลี่ยนเส้นทางไปที่ 127.0.0.1 ซึ่งหมายความว่าIP นี้จะใช้ในอุปกรณ์ของคุณเอง

เหตุใดการตั้งค่าการตรวจสอบการอัปเดตรายวันจึงเป็นสีเทา

การตรวจสอบอัตโนมัติสำหรับการอัปเดตใหม่จะใช้ได้เฉพาะเมื่อมีการติดตั้ง AdAway ไว้ในหน่วยความจำภายในเท่านั้น นี่เป็นเพราะข้อ จำกัด ของแพลตฟอร์มแอนดรอยด์

ทำไมถึงต้องรีสตาร์ทเพื่อให้มีผลเปลี่ยนแปลง?

Java บน Android มีแคช DNS ภายในของตัวเอง ระบบปฏิบัติการจะแสดงแฟ้ม hosts ใหม่ทันที (ตรวจสอบว่ามี ping ในบรรทัดคำสั่ง) แต่คุณจะต้องเริ่มต้นใหม่ Android เพื่อสร้างแคช DNS ของ Java

วิธีใช้เว็บเซิร์ฟเวอร์ใน AdAway

AdAway สามารถทำงานได้โดยปราศจากเว็บเซิร์ฟเวอร์

คุณสามารถเปิดใช้งานเว็บเซิร์ฟเวอร์ท้องถิ่นในลักษณะที่ AdAway ต้องการเพื่อตอบรับคำขอไปยังที่อยู่ IP ในเครื่อง 127.0.0.1. ซึ่งหมายถึงคำขอจากแอปพลิเคชันไปยังเซิร์ฟเวอร์โฆษณาที่ถูกเปลี่ยนเส้นทางไปยัง 127.0.0.1 ได้รับการตอบโดยเว็บเซิร์ฟเวอร์ของ AdAway.
แอพพลิเคชันบางตัวจะปฏิเสธการทำงานเมื่อเซิร์ฟเวอร์โฆษณาไม่สามารถเข้าถึงได้ ด้วยวิธีนี้พวกเขาจะสามารถเข้าถึงได้อีกครั้งตอบกลับด้วยหน้าว่าง ๆ และไม่มีภาพโฆษณา

ฉันสามารถ บล็อค/ปลดบล็อค โฮสต์เนม ที่เฉพาะเจาะจงได้อย่างไร?

เพิ่มชื่อโฮสต์ที่คุณต้องการบล็อกลงในบัญชีดำ ในรายชื่อของคุณ. นอกจากนี้คุณสามารถเพิ่มชื่อโฮสต์ที่คุณต้องการยกเว้นจากการบล็อกได้ที่ บัญชีที่ละเว้น และชื่อโฮสต์ที่คุณต้องการเปลี่ยนเส้นทางไปยังที่อยู่ IP ที่ระบุอยู่ใน รายชื่อที่เปลี่ยนเส้นทาง

ฉันสามารถหาแหล่งที่มาของโฮสต์ได้จากที่ไหน?

ดู รายชื่อแหล่งโฮสต์เพิ่มเติมจาก AdAway.
