由广告带来的收入

通常有一种误解认为如果用户从来不点击广告，那么屏蔽广告并不会给网站或者开发人员带来财务上的损失。然而，事实是仅仅展示广告也可以使开发人员获得收入。是否屏蔽广告取决于你。就我个人来说，我不会使用展示广告的应用，因为那让我很烦，所以没有一个屏蔽软件的话我是不会安装那些应用的

AdAway是如何工作？

AdAway 使用 hosts 文件 来阻止广告商的域名。hosts 文件位于 /system/etc/hosts ，用于将域名指向访问IP。这是一种不通过域名系统(DNS)将域名指向IP的传统拦截方式，所有不必要的域名重定向至 127.0.0.1 ，即将其指向设备自身。

为什么设置中的”每天检查更新“呈灰色无法选择？

自动检查更新只限于AdAway安装在内部存储。 这是一个安卓平台限制。

为什么我需要重启 Android 以使更改生效？

Android 上的Java 保留有内部自带的DNS缓存。新的Hosts文件将在操作系统立即生效(可在命令行上用ping验证)，但需要重启 Android 以重建 Java 的DNS缓存。

如何在AdAway里使用 webserver？

AdAway 也可不依赖Web网络服务器工作!

你可以在AdAway的偏好设置中启用本地网络服务器，让本地IP地址 127.0.0.1 响应请求。这意味着应用程序向广告商的链接请求将由AdAway的网络服务器重定向至 127.0.0.1 。
当广告商的服务器无法访问时，某些应用程序无法起作用。使用这种方法，他们将可访问，但只有空白页而没有广告图像。

如何封锁/解锁hostname?

在 你的名单 可添加你想阻止的域名到黑名单(阻止)。此外，可将想排除拦截的域名添加至 白名单 (不阻止) ，以及一些你想重定向至指定IP的域名添加至 重定向列表 (重定向)。
