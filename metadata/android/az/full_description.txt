Bloklayıcı hosts faylı dan istifadə edir
Hosts faylı IP ünvanları və host adlarının yönləndirilmələrinin siyahısını ehtiva edən sistem faylıdır.
Hər hansı tətbiq həmin host faylından reklam çağırarkən həmin çağırış heç bir şey etməyən yerli IP 127.0.0.1 ünvanına ötürülür.
Əvvəlcədən hazırlanmış hosts fayllarını yükləyə yaxud öz hosts fayllarınızı ağ və qara siyahılara atıb sistem faylına tətbiq edə bilərsiniz.
Yerli server açıb bloklanmış host adlarına cavab vermə və öz seçiminzə görə olan IP ünvanlarına yönlədirmə imkanlarınız da var.

Root vacibdir: Bəli. Hosts faylı standart halda yalnız oxuna bilən /system'də yerləşir.
