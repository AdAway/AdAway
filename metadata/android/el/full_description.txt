Μια εφαρμογή αποκλεισμού διαφημίσεων που χρησιμοποιεί το αρχείο hosts.
Το αρχείο hosts είναι ένα αρχείο συστήματος που περιέχει μια λίστα αντιστοίχισης μεταξύ host names και διευθύνσεων IP.
Όταν μια εφαρμογή ζητά μια διαφήμιση από host σε αυτό το αρχείο, το αίτημα αυτό ανακατευθύνεται στην τοπική διεύθυνση IP 127.0.0.1, η οποία δεν κάνει τίποτα.
Μπορείτε να κατεβάσετε τα προκαθορισμένα αρχεία hosts μέσα από την εφαρμογή για να τα ενσωματώσετε στο αρχείο hosts του συστήματος. Είναι επίσης δυνατή η χρήση των δικών σας αρχείων και η προσθήκη ορισμένων hosts σε λευκές και μαύρες λίστες.
Υπάρχουν επιλογές για να εκτελέσετε έναν τοπικό διακομιστή ιστού για να απαντήσετε σε αποκλεισμένα host names και να κατευθύνετε αιτήματα προς την διεύθυνση IP της επιλογής σας.

Απαιτεί Root: Ναι. Το αρχείο hosts βρίσκεται στο / system που είναι μόνο για ανάγνωση από προεπιλογή.
