Блокировщик рекламы использующий hosts файл.
Файл hosts это системный файл содержащий список сопоставлений между именами хостов и IP-адресами.
Когда приложение пытается запросить рекламу с хоста находящемся в этом файле, запрос перенаправляется на локальный IP 127.0.0.1, с которого ничего не загружается.
Вы можете загрузить предустановленные в приложении списки хостов, чтобы включить их в системный файл hosts. Вы также можете использовать ваши собственные списки и добавлять определенные адреса в белый и черный списки.
Имеется возможность запуска локального веб-сервера для перенаправления запросов к заблокированным адресам на выбранный вами IP-адрес.

Требуется root: Да. Файл hosts находится в разделе /system, который по умолчанию доступен только для чтения.
