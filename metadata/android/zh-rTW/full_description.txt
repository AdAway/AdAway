使用 hosts 檔案的廣告阻擋程式。
hosts 檔案是個包含在 host 及 IP 位址對應清單的系統檔案。
當 app 要求從清單中的 host 取得廣告時，此要求將會被轉址至本機 127.0.0.1 並不做任何事。
您可以在應用程式中下載預先定義的 hosts 檔案，以將它們合併到系統的 hosts 檔案中。您也可以使用自己的檔案，並將某些主機加入到白名單和黑名單中。
There are options to run a local web server to respond to blocked host names and to direct requests to the IP address of your choosing instead.
