Ένα πρόγραμμα αποκλεισμού διαφημίσεων δωρεάν και ανοιχτού κώδικα για Android.

Η εφαρμογή είναι διαθέσιμη για rooted και μη rooted συσκευές.
Χρησιμοποιώντας rooted συσκευή, η εφαρμογή ενημερώνει το αρχείο hosts του συστήματός σας που περιέχει μια λίστα αντιστοιχίσεων μεταξύ ονομάτων hosts και διευθύνσεων IP.
Χρησιμοποιώντας μη rooted συσκευή, η εφαρμογή χρησιμοποιεί τη δυνατότητα VPN για τον αποκλεισμό εξερχόμενων συνδέσεων με διαφημίσεις και προγράμματα παρακολούθησης.
Έτσι, όταν μια εφαρμογή ζητά μια διαφήμιση ή έναν ιχνηλάτη από έναν κεντρικό υπολογιστή σε αυτό το αρχείο, αυτό το αίτημα ανακατευθύνετε στην τοπική IP 127.0.0.1, η οποία δεν κάνει τίποτα.

Μπορείτε να κατεβάσετε προκαθορισμένες διαφημίσεις και λίστες αποκλεισμού ως hosts αρχεία από την εφαρμογή για ενσωμάτωση. Είναι επίσης δυνατό να χρησιμοποιήσετε τα δικά σας αρχεία και να προσθέσετε ορισμένα hosts σε επιτρεπόμενες και αποκλεισμένες λίστες.
Υπάρχουν επιλογές για να εκτελέσετε έναν τοπικό διακομιστή ιστού για να απαντήσετε σε αποκλεισμένα host names και να κατευθύνετε αιτήματα προς την διεύθυνση IP της επιλογής σας.
