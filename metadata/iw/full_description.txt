חוסם פרסומות חינמי ובקוד פתוח לאנדרואיד

זמין למכשירים בעלי הרשאות Root וגם ללא הרשאות.
במכשיר בעל הרשאות Root, היישום מעדכן את קובץ המארחים המובנה שלך שמכיל רשימה של מיפויים בין שמות מארחים וכתובות IP.
במכשיר שאינו בעל הרשאות Root, היישום משתמש בתכונת ה־VPN לצורך חסימת חיבורים יוצאים למודעות ולמעקבים.
לכן כאשר אפליקציה מבקשת מודעה או מעקב ממארח ​​בקובץ הזה, בקשה זו מופנית אל ה־IP המקומי – 127.0.0.1, שאינו עושה דבר.

באפשרותכם להוריד מודעות מוגדרות מראש ורשימות חסימה כקבצי מארח מתוך האפליקציה לשם שילובם. תוכלו אף להשתמש בקבצים משלכם ולהוסיף מארחים מסוימים לרשימות השונות.
There are options to run a local web server to respond to blocked host names and to direct requests to the IP address of your choosing instead.
