Бесплатный блокировщик рекламы с открытым исходным кодом для Android.

Приложение доступно для рутированных и нерутированных устройств.
Используя рутированное устройство, приложение обновляет файл hosts, который содержит список сопоставлений между именами хостов и IP-адресами.
Используя нерутированное устройство, приложение использует функцию VPN для блокировки исходящих соединений с сервисами рекламы и трекерами.
Поэтому, когда приложение запрашивает рекламу или трекер с хоста в данный файл, запрос приложения перенаправляется на локальный IP-адрес 127.0.0.1, который ничего не делает.

Вы можете загрузить предопределенную рекламу и списки блокировки в виде файлов hosts изнутри приложения для объединения. Также возможно использовать ваши собственные файлы и добавлять определенные хосты в списки разрешения и блокировки.
Имеется возможность запуска локального веб-сервера для перенаправления запросов к заблокированным адресам на выбранный вами IP-адрес.
