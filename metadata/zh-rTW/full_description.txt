開放原始碼的 Android 版廣告阻擋程式。

本應用程式可在已 root 與未 root 的裝置上使用。
如果使用已 root 的裝置，本應用程式會更新包含主機名稱與 IP 位址對應清單的系統 hosts 檔案。
如果使用未 root 的裝置，本應用程式會使用 VPN 功能來阻擋廣告與追蹤器的連線。
因此當應用程式從該檔案中的主機請求廣告或追蹤器，這個請求會被重新導向到本機 IP 127.0.0.1，什麼事都不會發生。

您可以在應用程式中下載預先定義的廣告阻擋器清單為 hosts 檔案。您也可以使用自己的檔案，並將某些主機加入到允許和阻擋清單中。
可經由選項建立本機網頁伺服器以針對被阻擋的主機名稱連線請求做出回應並重新導向至您自訂的IP 位址。
