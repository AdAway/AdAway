一個專為 Android 設計的免費且開源的廣告阻擋器。

此應用程式適用於已獲得 root 權限和未獲得 root 權限的裝置。
對於已獲得 root 權限的裝置，應用程式會更新您的系統 hosts 檔案清單，其中包含主機名稱與 IP 位址之間的映射。
對於未獲得 root 權限的裝置，應用程式會使用 VPN 功能來阻擋前往廣告和追蹤器的連線。
因此，當一個應用程式請求某個在該檔案中的主機提供廣告或追蹤器時，該請求會被重新導向至本機 IP 127.0.0.1，該 IP 不會進行任何操作。

您可以在應用程式內下載預先定義的廣告和阻擋清單，作為 hosts 檔案來整合使用。您也可以使用自己的檔案，並將特定的主機加入允許和被阻擋的清單。
此外，還有選項可以執行一個本機網頁伺服器，以回應被阻擋的主機名稱，並將請求重新導向到您選定的 IP 地址。
